Dentro da pasta com os arquivos do programa entre na pasta "build" e copie o endereço em que a pasta se localiza. Em seguida, abra o terminal e execute o comando "cd /caminho" para entrar na pasta build pelo terminal. Por fim, execute o comando "cmake .." e aguarda e comando rodar para depois inserir "make". Para executar o programa no Windows basta dar duplo clique no arquivo executável, caso esteja no Linux abra o terminal dentro da pasta "build" e execute "./Gerenciador". Para executar esses comandos é necessário ter o programa CMake instalado no computador.
